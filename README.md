# The Khoza Family's Last Brain Standing 🧠

A family word game — name countries, animals, fruits and more before time runs out.

---

## Deploy to Vercel (Free — 5 minutes)

### Option A — Drag & Drop (Easiest)

1. Go to **vercel.com** and create a free account
2. From your dashboard click **"Add New Project"**
3. Click **"Upload"** and drag this entire folder in
4. Vercel auto-detects Vite — just click **"Deploy"**
5. You'll get a live URL like `khoza-brain.vercel.app`

### Option B — GitHub (Recommended for updates)

1. Create a free account at **github.com**
2. Create a new repository called `khoza-brain`
3. Upload all these files to the repo
4. Go to **vercel.com**, sign in with GitHub
5. Click "Import" and select your repo
6. Deploy — done!

---

## Add to Home Screen (iPhone)

1. Open your live URL in **Safari**
2. Tap the **Share** button (box with arrow)
3. Tap **"Add to Home Screen"**
4. Name it **"Last Brain"** and tap Add
5. It now opens like a native app — full screen, no browser bar

---

## Features

- 7 categories: Countries, Vegetables, Fruits, Colours, Car Brands, Animals, Cities
- 2–8 players, pass and play
- Voice input (mic works fully on web)
- Brain facts after every correct answer
- Bragging rights + loser forfeit
- Hall of Fame — monthly, yearly and all-time champions
- Persistent records saved to device
